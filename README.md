# Projeto de Curriculo em HTML com Python

Este projeto gera um arquivo `index.html` de curriculo com base no perfil informado do LinkedIn.

## Arquivos

- `generate_resume.py`: script Python que monta e salva o HTML.
- `index.html`: arquivo final do curriculo (gerado automaticamente).

## Como usar

1. Abra o arquivo `generate_resume.py`.
2. Atualize os dados no dicionario `resume_data` com as informacoes do seu LinkedIn.
3. Execute o comando:

```bash
python3 generate_resume.py
```

4. Abra o arquivo `index.html` no navegador.

## Observacao

A extracao automatica de dados do LinkedIn pode ser bloqueada por restricoes da plataforma. Por isso, o projeto ja vem com uma estrutura pronta para voce preencher/ajustar rapidamente.
